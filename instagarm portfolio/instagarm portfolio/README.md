### Profile Picture
1. Add your profile picture to the `images/` folder
2. Name it `profile.jpg` (or update the filename in `index.html` line 32)
3. Recommended size: 300x300 pixels, square format

### Project Images
Add your project screenshots to the `images/` folder with these names:
- `project1.jpg` - E-Commerce Platform
- `project2.jpg` - Task Management App  
- `project3.jpg` - Mobile Banking App UI
- `project4.jpg` - Weather Dashboard
- `project5.jpg` - Brand Identity Design
- `project6.jpg` - Social Media Analytics

**Image Requirements:**
- Format: JPG, PNG, or WebP
- Size: 400x400 pixels (square format works best)
- File size: Under 1MB for faster loading

### Customizing Projects
Edit the `projects` array in `script.js` (lines 2-60) to:
- Update project titles and descriptions
- Change GitHub links to your repositories
- Add your live demo URLs
- Modify technology tags
- Add or remove projects
